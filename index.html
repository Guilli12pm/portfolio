Projects

Text Generation with QRNNs
Python

A project showcasing Text Generation using Quantum Recurrent Neural Networks with Free Fermionic Quantum circuits, Fully Quantum circuits, or Classical networks. With input text can be provided, the model will learn. Then any sentences can be provided and the model will predict the rest. This project used a collection of Shakespeare plays.

Sentiment Analysis with Category Theory and Lambeq
Python

This project shows how to perform sentiment analysis using a hybrid categorical/quantum approach where a sentence is converted to a network, which is converted to a quantum circuit, and the result depends on the measurement of that circuit.

Sentiment Analysis with QRNNs
Python

This project shows how to perform sentiment analysis using Quantum Recurrent Neural Networks. These networks are composed of Parametrised Quantum Circuits where the angle parameters are trained to produce a specific measurement. For a sentence, its negative or positive sentiment is produced. There are two datasets available, Tweets from Twitter and Rotten Tomatoes reviews.

Document Similarity with QRNNs
Python

This project shows how to predict document similarity using Quantum Recurrent Neural Networks. For two different sentences, the model predicts whether the meaning of the two sentences are shared, meaning they are similar, or whether they convey different meaning. This is based on the Quora dataset with pairs of questions of similar or different meaning

Image Classification with RNNs
Python

This project shows how to perform image classification using a Recurrent Neural Networks. Using the MNIST dataset, depending on the number drawn, a prediction of what that number represent is produced.
